FAQ

Here are some frequently asked questions for web developers.

  • How long does it take to create a website

    • This depends on a lot of things, such as the content, requested design, features, and many other factors that can drastically affect the timeframe of a project. For simple sites, a couple days is usually enough for a finished product. The larger a project is, the harder it is to say about a time frame.

  • How much does it cost to keep my website up

    • To have your website online you'll need two basic things. First off is a website domain. These are how people access your website. You can purchase them through various online stores, with a year cost that depends on the domain suffix (.com, .net, .us, etc). Second is the website hosting. Hosting is where your website's files are located for people to access. Hosting can be purchased from a lot of providers, which have plans that range from a month to a few years per payment.

  • What goes into creating a website

    • First off is an idea. Your idea is what creates the website, whatever it may be. With an idea, planning and design can take place, as well as studies on the user base. Using that information, development starts, optimizing the product with the user in mind.

  • What is Responsive Design

    • Responsive design is all about creating a webpage that looks good no matter what device you view it on. Not every visitor to your website will be using their computer, or their phone. Some may be using a tablet, others maybe a television. Responsive design accounts for all of the different widths and heights of devices and makes sure that the content of the website fits them perfectly.

  • What is this website built with?

    • My personal website is hand built with HTML, CSS, and Javascript. I wrote it using Sublime Text 2 on the Windows 8.1 OS. Hosting is provided thanks to Github Pages.
