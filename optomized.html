foodseeker.net

Work in Progress:

Foodseeker is a web-app that helps people decide where to eat. Too often people can't decide which restaurant they want to go to, sometimes there are too many choices and sometimes there are not enough. Foodseeker finds the user a restaurant using their query using HTML5 Geo-Location and the Yelp API. It is written using HTML5/CSS3 and Javascript. The site is fully responsive.
